Page de Test FunnelDoctor

Cette page permet de tester le tracking des visites et des clics via le snippet FunnelDoctor

Bienvenue sur notre page de démonstration

Cette page simule une landing page typique avec différents éléments interactifs qui déclencheront des événements de tracking.

Le script FunnelDoctor est installé sur cette page et capture les informations suivantes :

  • Les vues de page (automatiquement au chargement)
  • Les clics sur les boutons CTA
  • Les paramètres UTM présents dans l'URL

Tracking UTM

Le snippet capture les paramètres UTM et les stocke pendant 30 jours par défaut.

Suivi des clics

Tous les clics sur les éléments avec la classe .cta ou .buy-now sont automatiquement suivis.

Identification des visiteurs

Chaque visiteur reçoit un ID unique stocké localement pour un suivi cohérent.
